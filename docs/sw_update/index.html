SW Update

OTA and Ethernet-based firmware updates via FTP. Downloads remote files to local storage.

Overview

SW Update initiates an FTP download task. Server, credentials, remote path, and local path are passed at init. Result is shared via data bus.

API

AppSwUpdateInit(serverIp, port, user, pass, remotePath, localPath)
