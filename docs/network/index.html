Network Service

Manages GSM and Ethernet interfaces. PPP, lwIP. Multi-interface support with config-driven initialization.

Overview

Network Service initializes and starts GSM (PPP) and Ethernet (lwIP) interfaces. Configuration is per-customer; interfaces can be enabled/disabled.

  • GSM: PPP over UART, modem AT commands
  • Ethernet: ENC28J60 (SPI), lwIP stack

API

AppNetworkService_Start()
