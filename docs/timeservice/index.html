TimeService

Application time service with NTP, internal/external RTC, and software tick backends. JSON configuration and auto-generated wiring; unified API for time sync and format.

Overview

TimeService provides current time to other modules. Multiple backends (NTP, intRTC, extRTC, SoftTick) are selected via time_config.json. The generator script produces Cus_TimeService_Config.h and AppTimeService_Autogen.c.

  • Time zone support (e.g. UTC+3)
  • NTP sync with configurable server and period
  • Epoch / struct tm / string conversion

API

  • appTimeServiceInit
  • appTimeServiceGetTime
  • appTimeServiceGetEpoch
  • appTimeServiceFormatNow

Configuration

Customers/<customer>/Time/time_config.json

Keys: timeService.use, timeZone, ntp, intRTC, extRTC
